# 💳 Banking System (Console App)

A simple C# console application that models a banking system using Object-Oriented Programming (OOP) principles.  
The project demonstrates account management, transactions, and real-world business rules.

---

## 🚀 Features

- Create **Savings** and **Checking** accounts
- Deposit, Withdraw, and Transfer money
- **Overdraft support** for Checking accounts
- **Interest calculation** for Savings accounts (monthly/yearly)
- **Transaction history tracking**
- **Daily withdrawal limits**
- **Maximum balance limit (Savings)**
- **Bank fees**
  - Per-withdrawal fee
  - Overdraft fee (once per day)
- Generate account statements

---

## 🧠 Concepts Covered

- Object-Oriented Programming (OOP)
- Inheritance & Polymorphism
- Encapsulation
- Domain Modeling
- Exception Handling (custom exceptions)
- Collections (`Dictionary`, `IEnumerable`)
- Clean architecture separation (Models / Services / Exceptions)
